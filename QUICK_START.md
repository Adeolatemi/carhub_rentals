# CarHub Quick Start

## Frontend (Vite/React)
1. Terminal 1: `npm run dev` → http://localhost:5173

## Backend (Node/Prisma)
1. Terminal 2: `powershell -c "cd server; npm run dev"` → http://localhost:5000

## Prod Preview
1. `npm run build`
2. `npx serve dist -s` → auto-opens browser

## Test Flow
